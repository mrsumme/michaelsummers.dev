InfraRun

Intro

InfraRun integrates Richmond VAs 311 system with Strava to allow runners and cyclists to seamlessly request improvements by tagging Infrarun in their activities and attaching images of gaps in pedestrian/cycling infrastructure.

Tech

  1. Strava API
  2. AWS API Gateway
  3. AWS Lambda
  4. Selenium
